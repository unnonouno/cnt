=====
 cnt
=====

`cnt` counts words.


Requirement
===========

- gcc (w/ std=c++11)
- python (for build)


Build
=====

::

   $ ./waf configure
   $ ./waf
   $ sudo ./waf install


Usage
=====

::

   usage: cnt [options] ...

options:
  -m, --max     Maximum nubmer of words to show (unsigned long [=0])
  -?, --help    print this message
